# 🎯 Soc Ops – The Social Icebreaker Game

> **Break the ice and connect with people in a whole new way.** A modern, interactive bingo game designed for in-person mixers, conferences, and team events.

## What is Soc Ops?

Soc Ops transforms traditional icebreakers into an engaging, competitive experience. Players roam the room searching for people who match custom prompts—find someone who loves hiking, speaks three languages, or has visited all seven continents. Get five matches in a row (horizontally, vertically, or diagonally) and you win! 

Perfect for:
- **Networking Events** - Help attendees connect meaningfully
- **Team Building** - Break down silos and build camaraderie  
- **Conferences** - Make mixers memorable and fun
- **Mixers & Socials** - Spark genuine conversations

## 🚀 Getting Started

### Prerequisites
- [Node.js 22](https://nodejs.org/) or higher

### Installation & Development

```bash
# Clone and install dependencies
npm install

# Start the development server (hot reload enabled)
npm run dev
```

Visit `http://localhost:5173` and watch the magic happen!

### Build for Production

```bash
npm run build
```

The production build deploys automatically to GitHub Pages on every push to `main`.

---

## 🎮 How to Use

1. **Open Soc Ops** on your device
2. **Enter your name** to get started
3. **See your bingo board** with 25 randomized prompts
4. **Mingle with others** and get signatures/confirmations from people who match each prompt
5. **Get 5 in a row** (horizontally, vertically, or diagonally) to win!
